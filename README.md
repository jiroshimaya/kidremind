# セットアップ手順

以下の手順に従ってプロジェクトをセットアップし、実行してください：

1. サンプルデータをdataディレクトリにコピーします：
   ```
   cp -r sampledata data
   ```

2. .env_sampleファイルを.envにコピーします：
   ```
   cp .env_sample .env
   ```

3. 必要な依存関係をインストールします：
   ```
   pip install -r requirements.txt
   ```

4. アプリケーションを実行します：
   ```
   python app.py
   ```

これらの手順を完了すると、アプリケーションが起動し、実行される状態になります。
